# IPL-data-analysis-project
IPL match data analysis using python (2008-2019) with EDA and VISUALIZATION 
🏏 IPL Data Analysis Using Python

 Project Overview

This project analyzes Indian Premier League (IPL) match data from 2008 to 2019 using Python and Data Science techniques. The objective is to extract meaningful insights from historical IPL matches through data cleaning, analysis, and visualization.

---

 Objectives

- Analyze IPL match statistics.
- Identify the most successful teams.
- Study the impact of toss results on match outcomes.
- Analyze city-wise match distribution.
- Visualize trends using charts and graphs.

---

 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

 Dataset Information

- Dataset: IPL Matches Dataset
- Seasons Covered: 2008–2019
- Total Matches Analyzed: 756

---

 Key Findings

- Total IPL Matches: 756
- Mumbai Indians: 109 Wins
- Chennai Super Kings: 100 Wins
- Mumbai Hosted: 101 Matches
- Toss Winner Won Match: 393 Times
- Toss Winner Lost Match: 363 Times
- Average Winning Margin: 13.30 Runs

---

 Analysis Performed

- Data Cleaning
- Missing Value Analysis
- Season-wise Match Analysis
- Team Performance Analysis
- Toss Impact Analysis
- City-wise Match Analysis
- Winning Margin Analysis
- Data Visualization

 Project Files

 Conclusion

The project demonstrates how Python-based data analysis can be used to explore sports datasets and generate meaningful insights. Through visualization and statistical analysis, important IPL trends and patterns were identified.
